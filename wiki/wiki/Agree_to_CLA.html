Fork the misc-cla Repository

Go to https://github.com/EQdkpPlus/misc-cla and Fork the Repo

Create new CLA File

Go to your own forked misc-cla Repo (green Arrow), and create a new file (red arrow)

Edit new File

Give the new file the name of your GitHub account, with fileending .txt. As Content, add the CLA text from the README.md of the Repo.

Commit new file

After you are finished, commit the new file.

Add a pull Request

Click on pull Requests to start a new pull Request.
