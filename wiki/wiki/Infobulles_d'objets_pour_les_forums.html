Infobulles d'objets pour les forums

From EQdkp Plus
Jump to navigation Jump to search

Grâce à un petit script, il est possible d'utiliser les infobulles d'objets d'EQdkp Plus dans d'autres systèmes tels que des forums par exemple.

Installation

Insérez le code JavaScript suivant dans l'entête de votre forum. 

<script type='text/javascript' src='URL_DE_VOTRE_EQDKP/infotooltip/tooltips.php'></script>

Utilisation

Veuillez noter que depuis EQdkp Plus Version 1.0, la notation [item]ITEMNAME[/item] ne fonctionne plus.
Les infobulles pourront être utilisées sur n'importe quel site ou forum qui aura inclus le script évoqué plus haut. Pour indiquer au script quels éléments doivent être convertis en infobulle, il faudra englober le nom de l'objet dans une balise HTML de classe "EQdkpTooltip".

Exemple: 

<span class="EQdkpTooltip">Hache</span>

ou

<a href="viewitem.php?i=14" class="EQdkpTooltip">Hache</a>

BBCodes pour les Forums

Pour intégrer les infobulles à votre forum, vous devrez y créer des balises BBCode personnalisées (sans oublier d'inclure le script). Voici quelques exemples pour différents moteurs de forum.
